Creating a buy script:


  • Select a key on the virtual keyboard that you wish to bind something to:

 


  • Select as many items from the Buy Scripting tab:

 


  • Version 1.1.7 allows you to bind single clip increments of ammo. When single clips are selected, a drop down will appear which allows you to specify how many single clips you wish to purchase. Select the number of clips you want and press the green check mark in the upper right hand corner. Pressing the red X will cancel the single clip ammo selection.

  • Press the “Edit Item Sequence” button, or context menu option of the bind items, to bring up your binding sequence editor.  This arranges your items’ order.  Item order can be important in a buy script when funds are low.  CS & CZ will only purchase as much equipment as you have money, so put important items first, and luxury items last to ensure important items are bought when money is low.

  • Simply add  a custom bind or alias here and press enter or Add Bind to add it to your list of Current Bind Settings.


  • ADDITIONAL INFORMATION (changes from previous versions to 1.1.4 and up):
    • In previous versions of Bind Maker, certain items were combined.  Since the Colt (m4a1) is only available to CT’s and the Ak47 is only available to the T’s, these items used to be combined, but they are no longer done so.  If you want to ensure you get the right gun, you’ll want to select both weapons for your buy script: EX: check off Colt and AK so that you’ll get a colt when you’re on CT or Ak when you’re on T with the same button. The same thing goes for any T or CT specific weapons or items.
    • Previous versions would also add 2 flash grenade commands inline if you selected Flash(x2) / Flash Grenades (2 grenades), but in this version an alias is placed in your autoexec.cfg file (this file is created if none exists) for flash;flash; which is called double_flash; in CZ Bind Maker.  Removing this alias from your autoexec file will cause this bind created in CZ Bind Maker not to work.
